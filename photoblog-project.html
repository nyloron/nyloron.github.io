photoblog-project

This website was independently developed as a small project for a minor subject. The objective was to create a responsive website based on a self-devised concept (the presentation). Given that this was a minor subject, taught by the same instructors as the main computer science courses, the emphasis was on demonstrating a certain level of understanding of the topic. Consequently, there was limited time available for the actual realization of the task. The website was entirely hand-coded.

Attention: Embeded Elements on this page are resizeable!

project files

This presentation was crafted as a project guideline, intended to convey the fundamental concept. However, it ultimately diverged from an exact representation of the website. Navigation through the presentation is facilitated by hyperlinks, allowing users to explore various sections of the project with a simple click.
